Clone file from GitHub
1. cd desired location
2. git init			//create git repository
3. git clone "link"
4. git branch -a		//to see all branches
5. git checkout /*branch name*/ //to access branch

Check current branch in local
1. git branch

Pull file from GitHub
1. git branch MapReader 	//create a new branch called MapReader
2. git checkout MapReader	//change local branch to MapReader
3. git pull origin MapReader 	//pull from origin MapReader which is GitHub branch name

Push
1.	git status 			//check changes in all file
2.	git add /*file name*/		//add specific file before commit
2.1	git reset			//to cancel added file (revert back)
3.	git add .			//add all files before commit $ git commit -m "message"	
4. 	git commit -m "message"		//commit with a message	//commit with a message
5.	git push origin /*repo name */	//push file to repo in GitHub
